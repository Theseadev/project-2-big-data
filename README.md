# Project 2 - Big Data

## 📌 Deskripsi
Project 2 - Big Data merupakan proyek pengolahan dan analisis data skala besar menggunakan pendekatan Big Data.
Project ini bertujuan untuk membangun pipeline pemrosesan data yang terstruktur, efisien, dan dapat diskalakan.

## 🎯 Tujuan Project
- Mengimplementasikan konsep Big Data
- Membangun batch processing pipeline
- Mengolah data dalam jumlah besar
- Melakukan analisis data secara terstruktur

## 🛠️ Teknologi yang Digunakan
- Python
- PySpark
- Apache Spark
- WSL (Windows Subsystem for Linux)
- Git & GitHub

bigdata-project/

## 🚀 Cara Menjalankan Project

1. Aktifkan virtual environment
```bash
source venv/bin/activate
```

2. Jalankan script pipeline
```bash
python scripts/batch_pipeline_enterprise.py
```
